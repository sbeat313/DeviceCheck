# DeviceCheck 系統說明

本版新增「狀態切換通知」能力：

1. 每次探測設備後，比對探測前後狀態是否跨越正常/異常邊界。
2. 若從 `Alive` 變成非 `Alive`，或從非 `Alive` 變成 `Alive`，即建立通知事件。
3. 通知事件透過 `NotificationClient` 送往 `Notification.EndpointUrl`。
4. 通知接收人清單由 `Notification.Recipients` 統一設定。
5. 新增 `DeviceCheck.Notifier` 專案，作為接收通知 API 的 console/web 程式。

> 正常定義：`Alive`；異常定義：`Unknown`、`Busy`、`Dead`。
